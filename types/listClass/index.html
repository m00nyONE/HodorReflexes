    @param funcName - name of the function to run once

    listClass.Initialize

    function listClass.Initialize(listDefinition: table) ->  nil

    @param listDefinition - definition of the list

    NOT for manual use! this gets automatically called by :New() when creating a new list instance. initializes the list with the given definition. calls CreateSavedVariables(), CreateControls() and CreateFragment() once and deletes them afterwards. sets up event listeners for group changes, ui lock/unlock and player data updates.

    listClass.CreateCountdownOnControl

    function listClass.CreateCountdownOnControl(
  control: LabelControl,
  eventName: string,
  zeroTimerOpacity: number?
) ->  nil

    @param control - the control to render the countdown to

    @param eventName - the event name to register the countdown start callback to (must provide the following arguments: (unitTag, duration) where duration is in milliseconds)

    @param zeroTimerOpacity - optional opacity to set on the control when the timer reaches zero (default: self.sw.zeroTimerOpacity or 0.7)

    creates and registers a buff/debuff countdown timer on the control passed as argument. Can be used by custom themes as well. WARNING: DO NOT use it on controls that get recycled (e.g. playerRows)! ONLY USE on headers or static labels! Otherwise the timers will pile up and cause performance issues. They do NOT get automatically cleaned up on control recycling!
